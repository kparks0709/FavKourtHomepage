# Kourtney Parks – Favorites App (Mobile Web Application)

## Purpose
This project demonstrates my full mobile web application built with jQuery Mobile. It showcases my interests, includes navigation, and provides a simple mobile-friendly user experience.

## Site Map / Structure
- Home (index.html)
- About Me (about.html)
- Favorites (favorites.html)
- Favorites List (favorites-list.html)
- Contact (contact.html)

## Buttons

### Home Page
- **Open Panel**: Opens a side panel with quick navigation links.
- **About Me**: Takes the user to the About Me page.
- **My Favs / Favorites**: Takes the user to the Favorites page.
- **See Favorites / Favorites List**: Takes the user to the Favorites List page.
- **Contact Form**: Takes the user to the Contact page.
- **Rasmussen**: Opens the Rasmussen University website in a new tab.

### Panel (Navigation)
- **Home** → Loads the Home page.
- **About** → Loads the About Me page.
- **Favorites** → Loads the Favorites page.
- **Favorites List** → Loads the Favorites List page.
- **Contact** → Loads the Contact page.
- **Close** → Closes the navigation panel.
